# AcademiaTreineMais
Sistema criado pelo curso de Programador de Sistema, Senac Araguaina, na turma 2025.5.42 no ano de 2025.

## Descrição
Este sistema é voltado para uma academia pequena, para realizar :

- a organização dos horarios de funcionamento da academia,
- a escala dos personais trainers,
- a lista de alunos matriculado,
- os planos de assinaturas disponiveis,
- a construção da lista de treino para os alunos acessarem.

## Rquisitos

### Funcionais
- Gerar cadastro com informações pessoais, gerar forma de pargamento, listar infomações de agenda, listar frequencia
- Cadastrar o instrutor, relatar cliente, gerar documentos para contabilidade
- Infomar aparencia fisicas, gerar PDF, informar sobre o plano, editar clientes e funcionarios, armazenar historico de treino

### Não funcionais
- Segurança do aplicativo, restrição de idade, infomações modificaveis pelo instrutor ou cliente, acessibilidade para pessoas com daltonismo ou deficiencia tela escura ou tela clara

## Telas

### tela de login
- a tela de login ira servir para cadastrar informações basicas sobre o funcionario (numero de telefone, nome completo, CPF, curriculo de emprego)

### tela de rosto
- essa tela ira informar toda rotina do funcionario (cronograma)

### tela de termos de uso
- essa tela ira conter os termos de uso do aplicativo
  
### tela de registro final
- essa tela ira ter todas as informações do aplicativo como opções, agenda, tipo de treino, resgistro de presença, pagamento de plano emenur de plano.
